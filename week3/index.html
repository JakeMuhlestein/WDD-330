    Object Method: This

    “This” key word in java script OOP behaves unlike most other programming languages. It can be used in any function, even if it’s not a method of an object.
    Arrow functions are special: they don’t have their “own” this

    Ch5: Objects

    Object literals allow objects to be created quickly without the need for defining a class
    Access the properties of an object using the dot notation
    Access an object’s properties using bracket notation
    To call an object’s method use dot or bracket notation
    Loop through all of an object’s properties and methods by using a for in loop
    New properties and methods can be added/deleted to objects at any time in a program
    Object can contain other objects
    The parse() method takes a string of data in JSON format and returns a JavaScript object
    The stringify() method takes a JavaScript object and returning a string of JSON data
    Math has several math methods such as .exp(), .pow(), .sqrt(), and many more
    Date objects contain information about date and times

    Ch6: Document Object Model

    The Document Object Model (DOM) represents an HTML document as a network of connected nodes that form a tree-like structure
    The DOM treats everything on a web page as a node
    Node Code - 1:element, 2:attribute, 3:text, 8:comment, 9:body
    DOM Level 0 can still be employed
    getElementById() returns a reference to the element with a unique id attribute that is given as an argument
    getElementsByTagName() will return a live node list of all the elements with the tag name that is provided as an argument
    getElementsByClassName() will return a live node list of all elements that have the class name that is supplied as an argument
    document.querySelector() allows you to use CSS notation to find thefirstelement in the document that matches that matches a CSS selector provided as an argument
    getAttribute() method returns the value of the attribute provided as an argument
    setAttribute can change the value of an element’s attributes
    classList property is a list of all the classes an element has.
    createElement() method that takes a tag name as a parameter and returns that element
    A text node can be created using the document.createTextNode() method
    Elements can be removed from a page using the removeChild() method
    The replaceChild() method can be used to replace one node with another

    Ch7: Events

    Event listener are like setting a notification to alert you when something happens
    The click event occurs when a user clicks with the mouse, presses the Enter key, or taps the screen
    The addEventListener() method is called on a node object, the node to which the event listener is being applied
    Whenever an event handler is triggered by an event, the callback function is called
    There are also the mousedown and mouseup events. These both occurbeforea click event is fired
    There is the dblclick event, which occurs when the user doubleclicks on the element to which the event listener is attached
    Three events that occur when a user presses a key are: keydown , keypress and keyup
    Be careful when using the touchstart event as it fires as soon as a user touches the screen, touchend seems preferrable
    There are no 'swipe' events. These need to be created by using a combination of touchstart , touchmove , and touchleave
    An event listener can be removed using the removeEventListener()
